# EduElevate Frontend

Angular 20 SPA for the EduElevate education discovery platform.

## Stack

- Angular 20 (standalone components, signals)
- Tailwind CSS + Angular Material 3
- Lexend typography

## Prerequisites

- Node.js 20+
- Backend API running at `http://localhost:3000/api/v1`

## Setup

```bash
npm install
npm start
```

App: http://localhost:4200

Configure API URL in `src/environments/environment.ts` if needed.

## Scripts

| Command | Description |
|---------|-------------|
| `npm start` | Dev server |
| `npm run build` | Production build |

## Backend

Requires [eduelevate-backend](https://github.com/Bisham6/eduelevate-backend) for API data.
